Frontend Cheatsheet

CSS Properties per categorie
Categorie Property Uitleg
Layout display Bepaalt het layouttype van een element
Layout position Positioneert een element relatief of absoluut
Layout z-index Stapelvolgorde van elementen
Spacing margin Ruimte buiten het element
Spacing padding Ruimte binnen het element
Spacing gap Ruimte tussen items in flex/grid
Typografie font-size Grootte van tekst
Typografie font-family Lettertype
Typografie line-height Regelhoogte
Kleur color Tekstkleur
Kleur background-color Achtergrondkleur
Kleur background-image Achtergrondafbeelding
Overig border Rand rond element
Overig box-shadow Schaduw rond element
Animatie transition Overgangseffecten
Veelgebruikte HTML-tags
Tag Voorbeeld Uitleg
<div> <div>content</div> Container voor blokinhoud
<span> <span>text</span> Inline container
<a> <a href=\"#\">link</a> Hyperlink
<img> <img src=\"img.jpg\" alt=\"\"> Afbeelding
<input> <input type=\"text\"> Formulierveld
<button> <button>Klik</button> Knop
Flexbox Properties
Property Voorbeeld Uitleg
display display: flex; Activeert flex-container
flex-direction flex-direction: row; Richting van items
justify-content justify-content: space-between; Horizontale uitlijning
align-items align-items: center; Verticale uitlijning
flex-wrap flex-wrap: wrap; Laat items omvouwen
gap gap: 1rem; Ruimte tussen flex-items
